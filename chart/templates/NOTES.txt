Keycloak server "{{ .Values.keycloak.name }}" requested ({{ .Values.keycloak.instances }} instance(s)).

PREREQUISITE — the Keycloak Operator must already be installed cluster-wide.
This blueprint renders the Keycloak CR + Secrets but does NOT bundle the operator.
Install it once (per release version) with:

  kubectl apply -f https://raw.githubusercontent.com/keycloak/keycloak/{{ .Values.operator.version }}/operator/...keycloaks.k8s.keycloak.org-v1.yml
  kubectl apply -f https://raw.githubusercontent.com/keycloak/keycloak/{{ .Values.operator.version }}/operator/...keycloakrealmimports.k8s.keycloak.org-v1.yml
  kubectl apply -f https://raw.githubusercontent.com/keycloak/keycloak/{{ .Values.operator.version }}/operator/...kubernetes.yml

(See https://www.keycloak.org/operator/installation for the exact per-release URLs.)

  1. Wait for the server:  kubectl get keycloak {{ .Values.keycloak.name }} -o jsonpath='{.status.conditions}'
